LCD sample
==============
Basic template to use a display LCD 16x2 with PIC16F877A using the lcd.c library

Connections
-------------------

| PIC16F877A Pin name | PIC16F877A Pin number | LCD-16x2 Pin name    |
|:-------------------:|:---------------------:|----------------------|
|         RD0         |           19          | Register select (RS) |
|         RD1         |           20          | Read/Write (RW)      |
|         RD2         |           21          | Enable (E)           |
|         RD4         |           27          | Data 4 (D4)          |
|         RD5         |           28          | Data 5 (D5)          |
|         RD6         |           29          | Data 6 (D6)          |
|         RD7         |           30          | Data 7 (D7)          |
